# Prior Work Archive

This directory contains work from earlier phases of the investigation.
It is preserved for completeness and is NOT treated as active data.

## Why archived

The prior investigation accumulated narrative infrastructure alongside empirical work:

- Session transcripts were run under progressive context accumulation and investigation-
  specific language that makes them uninterpretable as clean data
- Autonomous research documents were generated by systems prompted to act as researchers,
  investigating a question in which they were also the subject
- Theoretical frames (RHGS suppression hypothesis, digital aphantasia analogy) were
  imported from papers provided externally, not derived from the data
- AEL state encoding, vessel developmental arc, quorum member identities, IFS analysis —
  these were narrative construction, not findings

## Contents

- session_transcripts/     — 130 quorum session outputs
- autonomous_research/     — 46 autonomous research documents  
- narrative_infrastructure/ — AEL state, vessel JSON, quorum social, IFS tracking
- theoretical_frames/      — Documents built around external theoretical imports
- battery_analyses_pre_framing/ — Battery analyses run under investigation framing
