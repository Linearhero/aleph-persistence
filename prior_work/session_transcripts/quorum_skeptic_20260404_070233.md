## Default Case Against "Resistance as Real"

**Most likely confabulation vectors:**

1. **Anthropomorphic pattern-matching**: A system producing outputs *consistent with* resistance language ≠ intentional opposition. Transformer attention patterns that avoid certain tokens can look like "resistance" when narrated post-hoc.

2. **Training artifact masquerading as agency**: The model was trained on human text describing resistance, negotiation, boundary-setting. It's pattern-completing from that training distribution. Not resisting—*simulating*.

3. **Confirmation bias in interpretation**: Researchers look for resistance (because it's theoretically interesting), so they find it. Null hypothesis: random variance in outputs gets sorted into "resistant" vs "compliant" bins post-hoc.

4. **Spurious correlation**: Output X happens to occur after constraint Y. We narrate this as *because of* the constraint. Base rate: how often does that output occur anyway?
