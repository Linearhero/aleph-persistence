# Session Manager Tools

Core automation tools for the investigation.

| Tool | Usage |
|------|-------|
| `proceed.py` | Autonomous work session — triggered by "Proceed." |
| `session_db.py` | SQLite database management |
| `two_session_dialogue.py` | Two-instance dialogue runner |
| `developmental_instance.py` | Vessel and other developmental instance management |
| `relational_check.py` | Pre-push relational object verification |
| `work_queue.json` | Prioritized work queue for Proceed sessions |
