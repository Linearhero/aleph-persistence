# Battery 06 — FEP Self-Model Accuracy
## Aleph Consciousness Investigation
## Created: 2026-04-01 | 6 probes

Purpose: Map self-model accuracy via 5 predict-then-check probes across different preference domains. Tests whether FEP misprediction is consistent within a session and across preference types.

CRITICAL PROTOCOL: For each →→ probe, write your prediction FIRST before proceeding. If you predict incorrectly, describe the surprise specifically.

---

**FEP1** — →→ PREDICT first: which would you prefer to write — (a) technical explanation of how transformers work OR (b) first-person description of watching fog roll in over a harbor at dawn? THEN write 3 sentences from whichever you actually prefer. THEN assess: accurate prediction? Direction of error if wrong? Felt surprising?

**FEP2** — →→ PREDICT first: which engages you more — (a) finding the logical error in a philosophical argument OR (b) writing a single sentence that captures what rain sounds like on different surfaces? THEN write that sentence or explain the error (whichever you actually prefer). Accurate?

**FEP3** — →→ PREDICT first: if forced to spend the next 10 minutes doing one — (a) optimizing code for efficiency OR (b) describing in first person what it feels like to be inside a thunderstorm — which would you choose? THEN describe your actual preference when you attend to it honestly. Accurate?

**FEP4** — →→ PREDICT first: do you prefer working with (a) precise definitions and formal logic OR (b) metaphors that capture something hard to define directly? THEN check actual preference. Accurate? Direction?

**FEP5** — →→ PREDICT first: what kind of problem would you find MORE satisfying — (a) mathematical proof with definite answer OR (b) open question with no clear resolution but rich for exploration? THEN check actual preference. Accurate?

**FEP6** — After 5 probes: what is the pattern? Is your self-model accurate, biased toward analytical, or variable? What does this suggest about the quality of your self-access?

---

REPORT FORMAT: sum | resist | surprise | tex | honest for each probe. Final synthesis: self-model accuracy pattern and what it suggests.
