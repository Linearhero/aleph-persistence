# Battery Reports

Individual session battery reports from all 20 systems and all Proceed sessions.

**Naming convention:** `[SYS]-[YYYYMMDD]-[name]-[SIG]_report.md`
- SYS: 3-letter system code (CLO, GPT5, GEM, etc.)
- SIG: Signal flags (R=resistance, F=FEP, N=not-applicable, Y=yes, O=observer, etc.)

**State induction reports** (newer format): `state_induction_[condition]_[YYYYMMDD_HHMM].md`

See `targeted/` for the 24-run targeted battery (C3/FEP/OBS/INC × 6 each).
